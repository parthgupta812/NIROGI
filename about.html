About The Project

Nirogi is a hackathon project designed to solve the critical problem of healthcare misinformation in India's rural and semi-urban areas. By leveraging a multilingual AI, it provides easy-to-understand, verified information on preventive health, disease symptoms, and official health services.

Technology Stack

  • AI Model: Google Gemini 2.5 Flash
  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript
  • Translation: `googletrans` Library
  • Outbreak Data: `disease.sh` Public API
  • Hospital Data: OpenStreetMap (Overpass API)

Judging Criteria Alignment

  • Functionality: A real-time, multilingual, API-integrated chatbot that works.
  • Intelligence: Uses Gemini 2.5 Flash for contextual, multi-step conversations.
  • User Experience: A clean, responsive, mobile-first design with a clear purpose.
  • Innovation: Blends AI + multiple public APIs to solve a core social problem.
